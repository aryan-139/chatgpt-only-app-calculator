YACA-Yet Another Calculator App

Functionalities:

YACA (Yet Another Calculator App) allows you to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. Simply enter the numbers and operators using the on-screen buttons or your keyboard, and the result will be displayed in the input field. All the basic operations with memory has been implemented, next, decimal support is implemented, with a super-attractive UI.

Developer:

YACA is developed by Aryan Raj for a basic starter project to be onboarded as a Volunteer Intern at Spartificial, using HTML, CSS, and JavaScript. It is built with the aim of providing a simple and user-friendly calculator experience.
